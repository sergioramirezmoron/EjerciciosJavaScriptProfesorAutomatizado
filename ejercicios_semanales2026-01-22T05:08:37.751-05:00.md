### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números
Define una función llamada `mapearNumeros` que tome dos argumentos: un array `arr` y una función `funcion`. La función debe aplicar la función proporcionada a cada elemento del array y devolver un nuevo array con los resultados de la función aplicada a cada elemento. Por ejemplo, si se pasa el array `[1, 2, 3]` y la función `x => x * 2`, la función debe devolver el array `[2, 4, 6]`.

#### Ejercicio 2: Combinar y Ordenar Arrays
Escribe una función llamada `combinarYOrdenar` que tome un número indeterminado de arrays con valores numéricos y un parámetro "creciente" o "decreciente". La función debe combinar todos los arrays en uno solo y ordenarlo según el parámetro proporcionado. Si no se proporciona el parámetro, la función debe ordenar el array en forma creciente por defecto.

#### Ejercicio 3: Diferencia y Unión de Arrays
Define una función llamada `diferenciaYUnion` que tome dos arrays `arr1` y `arr2`. La función debe devolver un objeto con dos propiedades: `diferencia` y `union`. La propiedad `diferencia` debe contener un array con los elementos que están en `arr1` pero no en `arr2`, y la propiedad `union` debe contener un array con todos los elementos de ambos arrays sin repetidos.
