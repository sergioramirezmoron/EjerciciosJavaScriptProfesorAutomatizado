### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números
Crea una función llamada `mapearNumeros` que tome un array de números y una función como argumentos. La función debe aplicar la función proporcionada a cada elemento del array y devolver un nuevo array con los resultados.

#### Ejercicio 2: Combinar y Ordenar Arrays
Escribe una función llamada `combinarYOrdenarArrays` que combine dos arrays de números y los ordene de forma creciente o decreciente según se indique. La función debe tomar como parámetros dos arrays y una cadena que indique el orden ("creciente" o "decreciente").

#### Ejercicio 3: Rotación y Verificación de Palíndromos
Crea una función llamada `rotarYVerificarPalindromo` que rote un array de caracteres hacia la derecha o izquierda y verifique si la cadena resultante es un palíndromo. La función debe tomar un array de caracteres y un número entero que indique los pasos de rotación. La función debe devolver `true` si la cadena resultante es un palíndromo y `false` en caso contrario.
