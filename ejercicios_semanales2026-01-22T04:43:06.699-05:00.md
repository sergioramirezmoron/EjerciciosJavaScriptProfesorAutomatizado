### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números con Condición
Crea una función que aplique la función `mapearNumeros` al array `[1, 2, 3, 4, 5]` con la función `function(x) { return x * 2; }` y luego filtre los resultados para incluir solo números mayores que 5.

#### Ejercicio 2: Combinar Objetos y Arrays
Utiliza la función `combinarObjetos` para combinar el objeto `{a: 1, b: 2}` con el array `[3, 4, 5]`. Luego, utiliza la función `combinarObjetos` nuevamente para combinar el resultado con el objeto `{c: 6, d: 7}`.

#### Ejercicio 3: Extraer Propiedades y Ordenar
Crea una función que tome un objeto como `{nombre: "Juan", edad: 25, ciudad: "Madrid"}` y un array de propiedades `["edad", "ciudad"]`, y utilice la función `extraerPropiedades` para extraer solo esas propiedades. Luego, ordene el objeto resultante por la propiedad "edad" utilizando la función `ordenarPorPropiedad`.
