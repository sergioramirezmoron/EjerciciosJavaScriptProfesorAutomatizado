### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números
Define una función `mapearNumeros` que tome un array de números y una función como argumentos. La función debe aplicar la función proporcionada a cada elemento del array y devolver un nuevo array con los resultados.

* Ejemplo de entrada: `mapearNumeros([1, 2, 3], function(x) { return x * 2; })`
* Ejemplo de salida: `[2, 4, 6]`

#### Ejercicio 2: Combinar Objetos
Escribe una función `combinarObjetos` que tome dos objetos como argumentos. La función debe combinar ambos objetos en uno solo y devolverlo.

* Ejemplo de entrada: `combinarObjetos({ a: 1, b: 2 }, { c: 3, d: 4 })`
* Ejemplo de salida: `{ a: 1, b: 2, c: 3, d: 4 }`

#### Ejercicio 3: Extraer Propiedades
Define una función `extraerPropiedades` que tome un objeto y un array de propiedades como argumentos. La función debe devolver un nuevo objeto que contenga solo las propiedades especificadas en el array.

* Ejemplo de entrada: `extraerPropiedades({ a: 1, b: 2, c: 3 }, ['a', 'c'])`
* Ejemplo de salida: `{ a: 1, c: 3 }`
