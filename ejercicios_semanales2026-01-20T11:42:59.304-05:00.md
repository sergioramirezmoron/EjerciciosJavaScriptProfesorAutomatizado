### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números con Función Personalizada
Crea una función llamada `mapearConCuadrado` que tome un array de números y devuelva un nuevo array con los cuadrados de cada número. Utiliza la función `mapearNumeros` para implementar esta funcionalidad.

#### Ejercicio 2: Combinar Objetos con Propiedades Comunes
Define dos objetos, `obj1` y `obj2`, con algunas propiedades comunes y algunas únicas. Utiliza la función `combinarObjetos` para combinar estos objetos en uno solo y muestra el resultado. Asegúrate de que las propiedades comunes se combinen correctamente.

#### Ejercicio 3: Extraer Propiedades y Ordenar por Promedio
Crea un objeto `alumnos` con propiedades como `nombre`, `edad` y `calificaciones` (un array de números). Utiliza la función `extraerPropiedades` para extraer las propiedades `nombre` y `calificaciones` de cada alumno. Luego, calcula el promedio de las calificaciones de cada alumno y ordena los alumnos por este promedio utilizando la función `ordenarPorPropiedad`. Muestra el resultado final.
