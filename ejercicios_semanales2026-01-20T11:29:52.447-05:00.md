### Ejercicios Prácticos de JavaScript
#### Ejercicio 1: Mapear Números con Condiciones
Crea una función que tome un array de números y una función como argumentos. La función debe aplicar la función proporcionada a cada elemento del array, pero solo si el elemento es mayor que 10. La función debe devolver un nuevo array con los resultados de la función aplicada a cada elemento que cumpla la condición.

#### Ejercicio 2: Combinar Objetos y Arrays
Escribe una función que combine un objeto y un array en un nuevo objeto. La función debe tomar un objeto y un array como argumentos. El array debe contener objetos con propiedades que se agregarán al objeto original. La función debe devolver el objeto combinado.

#### Ejercicio 3: Rotación y Ordenamiento de Arrays
Crea una función que rote los elementos de un array hacia la derecha o izquierda y luego ordene el array en orden creciente o decreciente. La función debe tomar un array, un número de pasos para la rotación y una cadena que indique el orden como argumentos. La función debe devolver el array rotado y ordenado.
