### Ejercicios Prácticos de Arrays en JavaScript
#### Ejercicio 1: Suma de Arrays con Restricciones
Escribe una función llamada `sumaArraysRestringida` que tome dos arrays `arr1` y `arr2` del mismo tamaño y que solo contengan números enteros positivos. La función debe devolver un nuevo array que contenga la suma de los elementos correspondientes de `arr1` y `arr2`, pero solo si la suma es menor o igual a 100. Si la suma de dos elementos es mayor a 100, el elemento resultante en el nuevo array debe ser 0.

#### Ejercicio 2: Duplicados con Orden Inverso
Define una función llamada `eliminarDuplicadosInverso` que tome un array `arr`. La función debe eliminar los elementos duplicados y devolver un nuevo array con elementos únicos, manteniendo el orden original, pero con los elementos en orden inverso al original.

#### Ejercicio 3: Filtrar Pares y Ordenar
Crea una función llamada `filtrarParesYOrdenar` que tome un array de números `arr` y devuelva un nuevo array que contenga solo los números pares, ordenados de mayor a menor. Además, la función debe incluir un mensaje que indique cuántos números pares se encontraron en el array original.
